지속적으로 웹서비스를 모니터링하고,
취약점을 파악하세요.

Embian APM e2eMon은 서비스 관점에서 Application을 관찰하고, 분석하며, 사용자의 액션을 기반으로 데이터의 흐름을 추적합니다.

더 이상 Software와 씨름하는데 시간을 낭비하지 마시고, 보다 더 훌륭한 Software를 만드는데 집중하세요.

e2eMon은 개발자가 찾기 힘든 장애 포인트를 추적하고 시각화 함으로써 개발자가 트러블슈팅 보다 새로운 코드를 작성하는데 더 집중할 수 있도록 도와줍니다.

  • 비지니스 기반 시스템 흐름 추적

    서비스 관점에서 전체 시스템을 바라보고 시스템을 관통하는 논리적인 비지니스 트랜젝션을 추적할 수 있습니다.

  • 코드 수준의 가시성 [개발예정]

    특정 코드나 SQL문이 성능에 영향을 미치는지 확인하기 위해 코드 수준의 Call Stack을 제공합니다.

  • 주요 트랜젝션 관리 [개발예정]

    비지니스 상 중요한 트랜젝션을 마킹해 두고, 별도의 뷰를 통해 응답시간, 호출건수, 에러율, 응답시간지연 등의 정보를 바로 확인 할 수 있습니다.

  • 트랜젝션 상세보기 [개발예정]

    같은 트랜젝션들의 추적 결과를 모아서 통계를 보여줌으로써, 트랜젝션 하나의 성능에 대한 깊은 통찰력을 얻을 수 있도록 도와드립니다.

"e2eMon은 서비스 전체를 관통하는 모니터링툴의 역할 뿐만 아니라, 문제 해결의 시작점으로써 웹 비지니스 지향 시스템의 일부가 되는 것을 목표로 합니다."

접속량이 폭주했을 때도, 코드 추가 후 장애가 발생했을 때도, e2eMon은 모든 상황을 빠짐없이 모니터링 하고 기록합니다.

성능 저하를 발생시킨 범인은 무엇이었을까요? 새로 추가된 코드? CPU? 아니면 Memory 부족?
e2eMon은 개발자와 운영자가 자신의 시스템이 실제 서비스 환경에서 어떻게 작동했었는지를 파악 할 수 있도록 도와줍니다.

  • 경보 시스템

    크리티컬 이슈 발생시 바로 경고 메세지를 E-mail이나 SNS를 통해 운영자나 개발자에게 전달 할 수 있으며, 경고 정책을 사용자가 직접 관리 할 수 있습니다.
    내용 더 보기  

  • 다양한 정보 제공

    Application의 성능을 쉽게 파악하고, 주요한 비지니스 트랜젝션을 쉽게 이해할 수 있도록 여러 형태의 Graph들과 기타 Visualization 기법들을 활용해 다양한 정보를 제공합니다.

  • 데이터 보안

    Embian APM, e2eMon은 고객사의 정보 보안을 위해 최선을 다하고 있습니다. 고객의 모든 데이터는 항상 안전하게 암호화하고 보호될 것이며, 상세한 보안 정책은 여기를 확인하시기 바랍니다.

특징 더 보기

웹 서비스를 위한 시스템 구조는 갈수록 복잡해지고 있으며, 그에 따라 시스템 관리는 갈수록 힘들어지고 있습니다.

e2eMon에서 제공하는 Server Map을 통해 외부API, 클라이언트, 데이터베이스, 서버같은 서비스를 구성하고 있는 복잡한 요소들을 시각적으로 보여줌으로써, 분산된 구성 요소간 상호 관계와 종속성을 쉽게 파악할 수 있으며, 이를 통해 서비스의 상태 및 품질을 직관적으로 모니터링 할 수 있습니다.

  • 구성 요소 자동 배치

    시스템을 구성하고 있는 요소들을 자동으로 파악하고 화면에 배치함으로써, 관리자의 할일을 최소화 합니다.

  • 전체 서비스 조망

    Server Map 하나만으로도 전체적인 시스템의 구성과 서비스 상태 및 취약점등을 한눈에 조망 할 수 있습니다.

  • 장애 위치 표시

    Server Map은 트랜젝션이 실패하거나 구성 요소간 연결이 끊기는 경우, 어디에서 문제가 발생했는지 바로 확인할 수 있도록 그 위치를 화면에 표시해 줍니다.

  • 실시간 모니터링 [개발예정]

    시스템의 상태를 실시간으로 모니터링 할 수 있도록 모니터링 관련 데이터는 모두 실시간으로 수집되고, 그 변화에 따라 화면도 자동으로 업데이트 됩니다.
